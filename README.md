# Minecraft Bedrock Superpowers — Pet Army

Tame **ANY** mob in Minecraft Bedrock Edition and build your own pet army!

## Features

- **Taming Wand** — Hit any mob to tame it (even hostile mobs like Zombies, Skeletons, and Creepers!)
- **Pet Follow AI** — Your pets follow you everywhere you go
- **Pet Defense** — When you get attacked, your pets rush in and fight for you
- **Pet Buffs** — Tamed pets get extra health, resistance, and strength
- **Call All Pets** — Sneak + use the wand to teleport all your pets to you
- **Release Pets** — Sneak + hit a pet with the wand to release it
- **Particle Trails** — Your pets have cool particle effects
- **Auto Wand** — New players automatically receive a Taming Wand on first join
- **Up to 10 pets** at a time

## How to Use

| Action | How |
|---|---|
| Tame a mob | Hit it with the Taming Wand |
| Release a pet | Sneak + hit it with the Taming Wand |
| Call all pets | Sneak + use the Taming Wand (right click / use) |
| Craft a wand | Diamond + Blaze Rod + Gold Ingot (vertical in crafting table) |

## Installation

1. Download both the **Pet Army BP** and **Pet Army RP** folders
2. Place them in your Minecraft Bedrock directories:
   - **Behavior Pack** → `com.mojang/development_behavior_packs/Pet Army BP/`
   - **Resource Pack** → `com.mojang/development_resource_packs/Pet Army RP/`
3. Create a new world (or edit an existing one)
4. Enable **"Beta APIs"** in the Experiments section
5. Add both packs to the world
6. Play!

## Can't Tame

- Players
- Ender Dragon
- Wither
- Non-living entities (armor stands, items, arrows, etc.)

## Crafting Recipe

```
    [ Diamond  ]
    [Blaze Rod ]
    [Gold Ingot]
```

Craft in a crafting table — the wand glows with enchantment shimmer!
